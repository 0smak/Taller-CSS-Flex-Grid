Propiedades Display más conocidas

hidden

El elemento actua como si no existiese (no se muestra)

block
  • Puedes añadirle width y height
  • Si no añades width, ocupa el 100% del padre
  • Cada elemento se situa en una nueva linea
  • Dentro podemos anidar otros elementos de tipo bloque o inline
<p>
<h1>
<div>
<header>
inline
  • No puedes añadirle width y height
  • No puedes añadirle margenes a la izquierda/derecha
  • Toman el espacio máximo del componente (no del padre)
  • Los elementos se situan en la misma linea
  • Dentro puede haber elementos de tipo inline
<span>
<a>
<strong>
<img>
inline-block

Esta propiedad es una mezcla de las dos anteriores, en la que podemos definirle un ancho y un alto, pero no tienen por que situarse en una nueva linea

<input>
<button>
<textarea>
<select>
